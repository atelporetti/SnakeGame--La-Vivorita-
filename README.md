# SnakeGame--La-Vivorita-
_Clasico juego de La Vivorita_

### Instalación 🔧

_1. Instalar las fuentes ubicadas dentro de la carpeta 'fonts'_

_2. Instalar pygame:_

      python3 -m pip install -U pygame --user
o si no funciona:

      pip pygame install

## Construido con 🛠️

* Python 🐍 v 3.9.1
* Librerias:
  * pygame
  * tkinter
